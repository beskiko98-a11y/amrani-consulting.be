Sites & applications sur mesure pour PME bruxelloises.

Sites vitrines, applications métier, automatisations internes. Je code avec un workflow assisté par IA (certifié Claude Code in Action par Anthropic) — vous gagnez en vitesse de livraison sans rogner sur la qualité ni la lisibilité du code.

Ce qui est inclus dans une mission Développement Web Bruxelles

  • Sites vitrines pour PME et indépendants : design moderne, performants, SEO-ready.
  • Applications métier sur mesure (gestion de stock, facturation, CRM léger, planning).
  • Intégrations API : Stripe, Mollie, Mailchimp, Google Workspace, etc.
  • Automatisations Google Apps Script, Zapier, Make pour fluidifier vos workflows.
  • Refonte de sites existants : passage de WordPress lourd vers du code propre.
  • Hébergement, déploiement, certificats SSL, configuration nom de domaine.
  • Maintenance évolutive après livraison, à la journée ou au forfait mensuel léger.
  • Code livré clair, documenté, transférable — vous restez propriétaire à 100%.

Types de projets web pour PME bruxelloises

  • Site vitrine sobre & rapide — comme celui que vous lisez, sans WordPress.
  • Outil interne de gestion — clients, stocks, planning, facturation.
  • API + intégration — connecter votre CRM avec votre comptable, etc.
  • Refonte WordPress lourde → site statique léger, hébergement 5€/mois.
  • Automatisation Apps Script — vos Google Sheets deviennent des outils intelligents.

Comment démarre une mission Dev Web ?

Appel de 30 min (gratuit) pour cadrer le projet : périmètre, maquettes éventuelles, budget. Devis au forfait sous 48h (préféré sur les projets cadrés) ou à la journée (préféré sur les missions évolutives). Premier rendu visible en 2-5 jours selon la complexité.
